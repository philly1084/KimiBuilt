Use this skill when the task involves feedback ingestion, backlog synthesis, experiment briefs, or release narratives.

## Workflow

Start from the supplied user evidence. Separate direct feedback from interpretation, group repeated themes, identify user impact, and turn the result into a concise backlog, experiment brief, or release narrative.

## Output

Return source-grounded themes, recommended next actions, open questions, and any assumptions that need validation.
