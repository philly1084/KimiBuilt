name: incident-report
description: Create a structured incident report from deterministic cluster evidence.
---

Use this skill when the task matches the description.

## Workflow

Sections: Summary, Impact, Detection, Timeline, Root-cause hypotheses, What changed, What worked, What did not work, Action items, Follow-up tests/evals. Do not invent evidence.

## Output

Provide clear evidence, safe next commands, and any missing information.
