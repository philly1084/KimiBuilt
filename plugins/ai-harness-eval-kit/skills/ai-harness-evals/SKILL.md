Use this skill when the task involves AI harness eval design, fixture coverage, prompt regression checks, model comparison, or harness run reporting.

## Workflow

Identify the behavior under test, define success criteria, inspect available fixtures, propose or run a focused regression slice, and report evidence before recommending changes.

## Output

Return eval intent, fixture coverage, pass/fail evidence, regressions or gaps, and the smallest useful next test.
